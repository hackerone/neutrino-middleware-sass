# neutrino-middleware-sass

Use SASS/SCSS with [neutrino](https://neutrino.js.org).

Install like so:
```bash
yarn add @izettle/neutrino-preset-sass
```

Then add it to your neutrino config in `package.json`:
```json
{
  "name": "example",
  "neutrino": {
    "use": [
      "@izettle/neutrino-preset-sass"
    ]
  }
}
```
